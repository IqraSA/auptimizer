Visualize one experiment result¶

aup.visualize is the entry point to visualize the result from an experiment eid:

python -m aup.visualize <eid>

Additional arguments¶

Usage: visualize.py [OPTIONS] EID

  Visualize result for experiment 

  Copyright (C) 2018  LG Electronics Inc. 

  GPL-3.0 License. This program comes with ABSOLUTELY NO WARRANTY; 

  Arguments:     eid {int} -- Experiment ID

Options:
  --log [debug|info|warn|error]
  --scale [linear|log]           Scale for y
  --save TEXT                    output file
  -h, --help                     Show this message and exit.
