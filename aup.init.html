Initialize experiment¶

See Configure HPO Algorithm for how to initialize the experiment search strategy.

Additional arguments¶

Usage: init.py [OPTIONS]

  Initialize experiment configuration for HPO 

  Copyright (C) 2018  LG Electronics Inc. 

  GPL-3.0 License. This program comes with ABSOLUTELY NO WARRANTY; 

  Raises:     Exception: If Auptimizer environment is not found.

Options:
  -e, --exp TEXT                  Experiment configuration file
                                  (experiment.json)

  -c, --aup TEXT                  Auptimizer Folder path
  -l, --log [debug|info|warn|error]
                                  Log level
  -o, --overwrite / -no, --no-overwrite
                                  overwrite existing file
  --reload / --no-reload          reload existing experiment configuration
  -h, --help                      Show this message and exit.
